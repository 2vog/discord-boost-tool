# How to use
 1. Insert your Capmonster Key in Settings.json.
 2. Put your Nitro Tokens in tokens.txt.
 3. Run Start.bat.
 4. Select 1 or 2 or 3 as it ask.
 5. Follow the Instructions and Enjoy.
